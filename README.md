# MQTT Live Monitor

Ein einfacher Web-Monitor für MQTT-Nachrichten mit Live-Ansicht, decodierten Daten von Chirpstack und Detailansicht.

## Erste Tests mir dem Liveserver

# Installation:

Repository klonen:

Wenn nicht vorhanden, dann git installieren:
```bash
apt install -y git
```
```bash
git clone https://github.com/BenAhrdt/mqtt-live-monitor.git
```
In das Verzeichniss des live-monitors wechseln
```bash
cd mqtt-live-monitor
```
Abhägigkeiten installieren

Wenn nicht vorhanden, dann npm installieren:
```bash
apt install -y nodejs npm
```
```bash
npm install
```
Anwendung starten
```bash
npm start
```
